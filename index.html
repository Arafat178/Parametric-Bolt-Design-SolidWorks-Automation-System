Copy this code and paste it into a SolidWorks Macro module to automatically generate the 3D part.

User Manual: How to Use in SolidWorks

  1. Input your parameters and click "Generate Spec", then "Generate SW Macro".
  2. Click "Copy Code to Clipboard".
  3. Open SolidWorks. Go to Tools > Macro > New.
  4. Save the macro file (e.g., AutoBolt.swp).
  5. The VBA Editor will open. Delete everything in the editor and paste the copied code.
  6. Press F5 (or click the Run icon) to execute the macro. The bolt will be generated instantly.
